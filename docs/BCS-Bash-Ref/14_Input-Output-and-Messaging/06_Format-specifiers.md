## 14.6 Format specifiers

Detailed reference.

- `%d`, `%i` тАФ signed integer.
- `%u` тАФ unsigned integer.
- `%o` тАФ octal.
- `%x`, `%X` тАФ hex (lower/upper).
- `%e`, `%E` тАФ scientific.
- `%f`, `%F` тАФ fixed-point float.
- `%g`, `%G` тАФ auto-format float.
- `%c` тАФ single character.
- `%s` тАФ string.
- `%b` тАФ string with `\` escapes interpreted.
- `%q` тАФ shell-quoted string.
- `%n$X` тАФ positional argument N (Bash 4.0+).
- `%(fmt)T` тАФ date/time.
- Flags: `-` left-align, `+` always sign, ` ` space-pad sign, `#` alternate, `0` zero-pad.
