## 15.10 Synopsis grammar

Notation for documenting CLI syntax.

- `[X]` — optional.
- `X|Y` — choice.
- `X...` — one or more.
- `[X...]` — zero or more.
- Uppercase: placeholder; lowercase: literal.
- Example: `cmd [-v] [-f FILE] {init|build|deploy} [ARGS...]`.
- Match the man-page convention for consistency.
