## 23.7 BSD `sh`

FreeBSD, OpenBSD, NetBSD use various `sh` implementations.

- FreeBSD: `ash` derivative.
- OpenBSD: `pdksh` (more capable; effectively ksh88).
- All POSIX-compliant; few extensions.
- For cross-BSD portability, restrict to POSIX.
